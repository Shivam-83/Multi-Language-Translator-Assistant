# Multi-Language Translator Assistant

A Python application that translates text into multiple languages using the Google GenAI SDK powered by Gemini 2.5 Flash.

## Supported Languages
- Hindi
- Telugu
- French

## Setup

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure API Key**:
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

## Usage
Run the script to see the translation:
```bash
python app.py
```
